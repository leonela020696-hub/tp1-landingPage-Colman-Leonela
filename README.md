Aclaraciones respecto a los repositorio para los trabajos de Enstandares en Desarrollo Web.

Para poder visualizar el proyecto, les detallo las siguientes opciones:

1) Se puede clonar este repositorio en una carpeta local y alli apareceran cada archivo html que al abrirlo se podra visualizar en el navegador habitual del nuestro ordenador

2) Si descargamos y abrimos el proyecto con el editor de codigo Visual Studio Code, y a su vez descargamos en VSC la extencion "Live Server" podremos visualizazr la pagina web y todos los cambios que hagamos
   en la misma en nuestro navegador por defecto, al realizar cada cambio lo iremos guardando y actualizando el navagador los podremos ver al instante en la página web.
   
3) La tercera opcion seria clonar el repositorio, abrir github en la web, luego ir a la pestaña "Settings", luego "Pages" para activar el servicio de hosting y alli poder visualizarla correctamente.
